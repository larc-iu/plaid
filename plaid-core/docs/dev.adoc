= Development

* `clojure -X:dev`: start development server by typing `(start)`. See <http://localhost:8085>
* `clojure -M:test`: run tests
* `clojure -M:outdated`: find outdated dependencies
* `clojure -X:uberjar`: compile to a single `.jar` for production
* `clojure -M:nrepl`: start nREPL server (for Clojure MCP)

NOTE: Client libraries are maintained separately in `plaid-client-js/` and `plaid-client-py/`.

== OpenAPI Support
* See playground at <http://localhost:8085/api/v1/docs/index.html#/>
* See JSON at <http://localhost:8085/api/v1/openapi.json>

== Database Access (pgwire)
XTDB v2 exposes a PostgreSQL wire-compatible interface on port 5433.
You can connect with `psql` or any PostgreSQL-compatible client such as DataGrip:

[,bash]
----
psql -h localhost -p 5433 -d xtdb
----

In DataGrip, create a new PostgreSQL data source with host `localhost`, port `5433`, and database `xtdb` (no username/password required).

NOTE: Column names for nested keys use the `$` separator (e.g. `project$name`). This interface is read-only from the application's perspective.
